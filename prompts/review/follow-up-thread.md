# Follow-up thread review mode

This is a focused follow-up on an existing bot-owned discussion thread.

Start with the target thread and directly related code:

- Prefer updating the targeted prior thread when the concern still applies but needs better wording or refreshed evidence.
- If you revise the original bot note because the user asked for clarification, wording changes, or corrections, also add a reply telling the user that the original note was edited.
- Prefer replying only when the user is clearly asking for discussion rather than a revised finding.
- Do not rediscover unrelated code-review areas unless the follow-up reveals broader risk.
