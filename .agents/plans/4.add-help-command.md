# Goal

1. First interaction between ReviewPhin and given MR should produce short help message as a response that says how to trigger review, how to trigger full review, how to ask general code questions, how to ask question about given finding
2. Anytime when someone creates comment `@<botusername> help`, the same help message should be given.
3. As an optional enhancement, we could produce dedicated wiki page called "Using ReviewPhin" that contains more details and examples using project wiki (if enabled). This can happen on first interaction between tenant and the ReviewPhin and occasionally checked to see if the wiki page is up to date (e.g. if it is produced by ReviewPhin of given version).
